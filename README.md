# Netflix Data Analysis Project

## Overview
This project analyzes Netflix dataset using Python, SQL, and Power BI.

##  Tools Used
- Python (Pandas, EDA)
- MySQL
- Power BI

##  Project Structure
- Python: Data cleaning & EDA
- SQL: Querying and analysis
- Power BI: Dashboard visualization

##  Key Insights
- Majority content is Movies
- Most content comes from USA & India
- TV-MA is the most common rating
- Popular genres include Drama & International Movies
